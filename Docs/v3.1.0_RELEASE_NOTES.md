# Live Stopwatch v3.1.0

## 主な変更

- NOW PLAYING／SELECTEDラベルを常時表示
- 全画面表示および大きなウィンドウで、曲番号・曲名・ラベルを比例拡大
- タイマーと情報表示の比率を維持するレスポンシブレイアウトへ刷新
- 長い曲名を画面幅に合わせて自動縮小
- 小型ウィンドウでの最低可読サイズを維持
- Remote Scriptとアプリのバージョンを3.1.0へ統一

## 表示設計

ウィンドウの幅と高さから共通スケールを計算し、以下を連動して拡大縮小します。

- NOW PLAYING / SELECTEDラベル
- 現在再生中・選択中の番号と名称
- タイマー
- ステータスバーと設定アイコン

タイマーだけが過度に拡大し、曲名が小さいままになる問題を改善しています。

## 互換性

- macOS 11 Big Sur以降
- Intel Mac / Apple Silicon Mac
- Xcode 12.4
- Swift 5.3
- Ableton Live 12
