# v2.30 Stability Release

## 方針

v2.6の機能と画面構成を維持し、macOS 11 Big Sur /
Xcode 12.4との互換性修正だけを行った安定化版です。

## 修正

- `Text.textSelection(.enabled)`を削除
- macOS 12専用APIが残っていないかソースを再確認
- Settings画面のVersion表記を2.30へ更新
- Embedded Remote Scriptとインストーラーのバージョンを2.30へ統一
- Remote ScriptのPython構文チェックを実施

## 変更していないもの

- 時計表示
- CPU負荷対策
- STOP処理
- SELECTED処理
- Remote Scriptアプリ内インストール
- MIDI設定
- 統合設定画面
- ウィンドウレイアウト

## 確認順

1. Xcode 12.4でCommand + B
2. Command + Rで起動
3. 歯車から設定画面を開く
4. Remote Scriptをインストール／更新
5. Ableton Liveを完全終了して再起動
6. STOP、SELECTED、CPU負荷を確認
