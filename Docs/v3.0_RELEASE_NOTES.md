# Live Stopwatch v3.0.0

正式リリース。

## 主な変更
- General設定を次回起動時に復元
- MIDI入力ポートを次回起動時に自動復元・接続
- メインウィンドウのサイズと位置を保存
- 初回セットアップウィザードを追加
- 初回起動時にRemote Scriptを自動インストール
- Ableton Live／MIDI設定説明を追加
- START・STOP・Remote Script接続テストを追加
- AboutとSIGNAL FLOWブランドロゴを追加

## 対応
- macOS 11以降
- Intel Mac / Apple Silicon Mac
- Ableton Live 12 Standard
