# Live Stopwatch v3.0 Test Plan

## First launch
- Remote Script installs automatically.
- Setup wizard opens only on the first launch.
- Ableton Live and MIDI instructions are readable.
- Completing the wizard prevents it from reopening.

## Persistence
- Always on Top is restored after relaunch.
- STOP reset preference is restored after relaunch.
- Selected MIDI source is restored and reconnects.
- Main window position and size are restored.

## Connection test
- Remote Script indicator becomes green after Live sends data.
- START indicator becomes green after MIDI Start.
- STOP indicator becomes green after MIDI Stop.

## Playback
- MIDI Start always resets to 0:00 and starts.
- MIDI Stop resets or pauses according to the saved preference.
- Scene changes reset and start the timer.
- NOW PLAYING and SELECTED remain correct.

## Compatibility
- Intel Mac / macOS 11
- Apple Silicon Mac
- Ableton Live 12 Standard
- Long-running CPU check: 60 minutes
