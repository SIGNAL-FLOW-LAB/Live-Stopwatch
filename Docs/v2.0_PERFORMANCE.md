# v2.0 Performance Update

## 主な低負荷化

- 時計更新を50 Hzから10 Hzへ変更
- 画面表示の0.1秒精度は維持
- 表示値が変わらない場合は`@Published`更新を行わない
- MIDIパケット解析から`Mirror`を削除
- MIDI Clock受信中のUI更新を状態変化時だけに限定
- 状態監視を0.5秒から1秒へ変更
- Remote Script通信はイベント受信時のみ更新

## 目標

Intel Macで通常時CPU 5%未満を目標とします。
実際の数値はMacの世代、Live Set、画面サイズによって変動します。

## UI

- macOSタイトルバーにアプリ名を表示
- SIGNAL FLOW用アプリアイコンを追加
