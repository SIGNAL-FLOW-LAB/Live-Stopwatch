# v2.21 Release Notes

## 時計表示

- ウィンドウサイズに応じて時計フォントを連続的に自動調整
- compact境界で時計サイズが急変する問題を修正
- 幅と高さの両方に収まる最大サイズを計算
- AppKit軽量描画は継続

## SELECTED初期表示

- 初回起動時は必ずMain列のScene名を表示
- Liveが保持しているTrack 1選択を初回表示に使用しない
- 2回目以降は、実際にハイライトされた通常トラックのClip Slotと
  選択Sceneが一致した場合のみクリップ名を表示

## 必須

アプリ内Remote Script設定からv2.21へ更新し、
Ableton Liveを完全終了して再起動してください。
