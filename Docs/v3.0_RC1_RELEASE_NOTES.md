# Live Stopwatch v3.0 RC1

## UI

- アプリ名を`Live Stopwatch`へ統一
- 起動時のみ450 × 280
- 起動後は自由にリサイズ可能
- Aboutから`by SIGNAL FLOW`を削除
- 透明背景の大型ストップウォッチアイコンへ変更

## 維持した機能

- macOS 11 / Xcode 12.4対応
- Intel / Apple Silicon Universal Binary設定
- Remote Scriptアプリ内インストール
- MIDI / Script / Clock / Play / Stop表示
- NOW PLAYING / SELECTED表示
- 常に最前面設定
- 10 Hz時計更新と多重起動防止

## RCテスト

1. Xcode 12.4でビルド
2. 起動時サイズが450 × 280であること
3. ウィンドウを自由に拡大・縮小できること
4. About表記とFinder名が`Live Stopwatch`であること
5. Remote Scriptを更新後、STOPとSELECTEDを確認
6. Intel Macで2時間連続CPUテスト
