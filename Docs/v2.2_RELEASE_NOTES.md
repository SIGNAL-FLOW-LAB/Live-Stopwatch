# v2.2 Release Notes

## CPU負荷改善

- 時計表示をSwiftUI TextからAppKit NSTextFieldへ変更
- 0.1秒ごとの更新で画面全体を再描画しない設計へ変更
- SwiftUIへ公開する状態を再生／停止などの変化時に限定
- MIDIControllerの多重初期化を防止
- Timer、UDP Listener、CoreMIDIの重複起動を防止

## SELECTED表示修正

アプリ起動後、Main列のSceneを最初に選択したときに、
Track 1の古いクリップ名がSELECTEDへ表示される問題を修正しました。

v2.2では以下の条件をすべて満たす場合のみクリップ名を表示します。

1. 選択トラックが通常トラック
2. 選択Sceneと同じ位置のClip Slot
3. Liveのhighlighted_clip_slotと完全一致
4. そのスロットにクリップが存在

条件に一致しない場合はMain列のScene名を表示します。

## 必須作業

Remote Scriptがv2.2へ更新されています。

アプリ内のRemote Scriptセットアップから「更新」を実行し、
Ableton LiveをCommand + Qで完全終了して再起動してください。

## Intel Macテスト目標

- 停止中：1〜2%
- カウント中：10%未満を目標
- 2時間後もCPU使用率が上昇し続けないこと
