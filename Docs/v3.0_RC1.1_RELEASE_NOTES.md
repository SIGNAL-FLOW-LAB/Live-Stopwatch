# Live Stopwatch v3.0 RC1.1

RC1の設定画面を整理した軽微なUI・仕様修正版です。

## Changes

- 設定画面左上のアイコンを、アプリ本体のアイコンと統一
- 「MIDI STARTで0へリセット」の設定項目を削除
  - MIDI START受信時の0:00リセットと開始は、アプリの固定動作として維持
- 「常に最前面」を「常に最前面に表示」へ変更
- 「MIDI STOPで0へリセット」を「STOP受信時にタイマーを0:00へ戻す」へ変更
- Aboutの表示を `Version 3.0 RC1.1` へ更新

## Compatibility

- Xcode 12.4
- Swift 5.3
- macOS 11.0以降
- Intel Mac / Apple Silicon（Universal Binary）
