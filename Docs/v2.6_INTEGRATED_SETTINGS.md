# v2.6 Integrated Settings

## 開き方

メイン画面右下の歯車を押すと、統合設定画面が開きます。

## 初回起動

以下の場合は設定画面を自動表示します。

- v2.6の設定画面をまだ開いていない
- Remote Scriptが未インストール
- Remote Scriptの更新がある

## 設定画面

- General
  - 常に最前面
  - MIDI START/STOPリセット
- Remote Script
  - 状態表示
  - インストール／更新
  - 保存場所表示
  - Live接続中Scriptバージョン
- MIDI
  - 入力選択
  - 接続
  - 再スキャン
- About
  - アプリ名
  - Version 2.6
  - Copyright
