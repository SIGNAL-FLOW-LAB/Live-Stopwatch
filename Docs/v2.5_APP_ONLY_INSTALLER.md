# v2.5 App-only Remote Script Installer

## 変更内容

- `Remote Scriptをインストール.command`を完全廃止
- 外部の`RemoteScript`フォルダを配布物から削除
- Remote Scriptはアプリ内Resourcesへ同梱
- インストール・更新・上書きをSwiftのFileManagerだけで実行
- Terminal、AppleScript、shell、`.command`は使用しない
- Intel Mac / Apple Silicon共通

## 操作

1. アプリを起動
2. 初回セットアップ、または歯車から`REMOTE SCRIPT`を開く
3. `インストール`または`更新`を押す
4. Ableton LiveをCommand + Qで完全終了
5. Ableton Liveを再起動
6. Tempo & MIDIで`SIGNAL_FLOW_Clip_Watcher`を選択

## インストール先

```text
~/Music/Ableton/User Library/Remote Scripts/
SIGNAL_FLOW_Clip_Watcher
```

## 注意

User Libraryの保存場所をAbleton Liveで独自変更している場合、
v2.5は標準のUser Libraryへインストールします。
